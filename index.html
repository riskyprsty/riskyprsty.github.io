About Me

Maaf karena masih sedikit yang dapat saya ceritakan mengenai diri saya oleh karena itu, pada section ini hanya akan membahas mengenai website ini sendiri.

Halaman ini dibuat menggunakan framework Bootstrap versi 5.3. Beberapa component yang digunakan diantaranya Navbar, Card, Button, dll. Kustomisasi dilakukan dengan menggunakan utilities/class yang sudah disediakan oleh bootstrap serta ada beberapa kustomisasi yang menggunakan CSS murni (didalam file style.css) dikarenakan keterbatasan kustomisasi jika hanya menggunakan class yang disediakan Bootstrap (jika tanpa menggunakan SASS).
